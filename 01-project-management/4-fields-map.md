## How to Read This Map

Each row maps a task across four fields:

1. **Phase / What** — the deliverable (from the WBS)
2. **Who** — the named individual responsible
3. **When** — the target completion week
4. **Standard / Quality Criterion** — what "done" looks like; how the group will judge this task complete

This document should be updated at each formal weekly meeting to reflect actual status.
